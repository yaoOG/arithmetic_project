
- **主键索引 (Primary Key)**：
    
    - 一张表只能有一个，极其严格。要求字段值必须唯一，且**绝对不能为 NULL**。
        
- **唯一索引 (Unique Index)**：
    
    - 要求字段的值必须唯一，但**允许存在多个 NULL 值**。主要用于业务防重（比如保证用户的手机号不能重复）。
        
- **普通索引 (Normal/Index)**：
    
    - 最基础的索引，没有任何约束，唯一的作用就是加快查询速度。允许数据重复，允许为 NULL。
        
- **联合索引 / 复合索引 (Composite Index)**：
    
    - 把表里的多个列组合起来共同创建一个索引（比如 `INDEX(name, age, gender)`）。
