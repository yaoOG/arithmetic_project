# 高频题型模板

## 一、数组和双指针

### 1. 快慢指针删除重复

适用：

- 有序数组原地去重。
- 原地删除指定元素。

模板：

```java
int slow = 0;
for (int fast = 0; fast < nums.length; fast++) {
    if (应该保留 nums[fast]) {
        nums[slow++] = nums[fast];
    }
}
return slow;
```

### 2. 两端指针

适用：

- 有序数组两数之和。
- 盛最多水的容器。
- 回文判断。

模板：

```java
int left = 0, right = nums.length - 1;
while (left < right) {
    int sum = nums[left] + nums[right];
    if (sum == target) {
        return true;
    } else if (sum < target) {
        left++;
    } else {
        right--;
    }
}
return false;
```

## 二、滑动窗口

### 1. 最长窗口

适用：

- 最长无重复子串。
- 最多包含 K 种字符。

模板：

```java
int left = 0, ans = 0;
Map<Character, Integer> window = new HashMap<>();

for (int right = 0; right < s.length(); right++) {
    char c = s.charAt(right);
    window.put(c, window.getOrDefault(c, 0) + 1);

    while (窗口不合法) {
        char d = s.charAt(left++);
        window.put(d, window.get(d) - 1);
        if (window.get(d) == 0) {
            window.remove(d);
        }
    }

    ans = Math.max(ans, right - left + 1);
}
```

### 2. 最短窗口

适用：

- 最小覆盖子串。
- 长度最小的子数组。

模板：

```java
int left = 0, ans = Integer.MAX_VALUE;

for (int right = 0; right < n; right++) {
    // 加入 right

    while (窗口满足条件) {
        ans = Math.min(ans, right - left + 1);
        // 移出 left
        left++;
    }
}
```

## 三、二分查找

### 1. 普通二分

```java
int left = 0, right = nums.length - 1;
while (left <= right) {
    int mid = left + (right - left) / 2;
    if (nums[mid] == target) {
        return mid;
    } else if (nums[mid] < target) {
        left = mid + 1;
    } else {
        right = mid - 1;
    }
}
return -1;
```

### 2. 左边界

```java
int left = 0, right = nums.length;
while (left < right) {
    int mid = left + (right - left) / 2;
    if (nums[mid] >= target) {
        right = mid;
    } else {
        left = mid + 1;
    }
}
return left;
```

### 3. 答案二分

适用：

- 最小化最大值。
- 最大化最小值。
- 运载能力、吃香蕉速度、分割数组。

模板：

```java
int left = 最小可行值, right = 最大可行值;
while (left < right) {
    int mid = left + (right - left) / 2;
    if (check(mid)) {
        right = mid;
    } else {
        left = mid + 1;
    }
}
return left;
```

关键：

```text
check(mid) 必须具备单调性。
```

## 四、链表

### 1. 虚拟头节点

适用：

- 删除节点。
- 合并链表。
- 分隔链表。

模板：

```java
ListNode dummy = new ListNode(0);
dummy.next = head;
ListNode cur = dummy;

while (cur.next != null) {
    if (需要删除 cur.next) {
        cur.next = cur.next.next;
    } else {
        cur = cur.next;
    }
}

return dummy.next;
```

### 2. 快慢指针

适用：

- 找中点。
- 判断环。
- 找倒数第 K 个。

模板：

```java
ListNode slow = head, fast = head;
while (fast != null && fast.next != null) {
    slow = slow.next;
    fast = fast.next.next;
}
```

## 五、树 DFS / BFS

### 1. 递归 DFS

```java
void dfs(TreeNode root) {
    if (root == null) {
        return;
    }
    // 前序位置
    dfs(root.left);
    // 中序位置
    dfs(root.right);
    // 后序位置
}
```

### 2. 层序遍历

```java
Queue<TreeNode> queue = new ArrayDeque<>();
queue.offer(root);

while (!queue.isEmpty()) {
    int size = queue.size();
    for (int i = 0; i < size; i++) {
        TreeNode node = queue.poll();
        if (node.left != null) queue.offer(node.left);
        if (node.right != null) queue.offer(node.right);
    }
}
```

## 六、回溯

### 子集/组合模板

```java
void backtrack(int start, List<Integer> path) {
    result.add(new ArrayList<>(path));

    for (int i = start; i < nums.length; i++) {
        path.add(nums[i]);
        backtrack(i + 1, path);
        path.remove(path.size() - 1);
    }
}
```

### 排列模板

```java
void backtrack(List<Integer> path, boolean[] used) {
    if (path.size() == nums.length) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        used[i] = true;
        path.add(nums[i]);
        backtrack(path, used);
        path.remove(path.size() - 1);
        used[i] = false;
    }
}
```

## 七、动态规划

### 1. 一维 DP

```java
int[] dp = new int[n + 1];
dp[0] = 初始值;

for (int i = 1; i <= n; i++) {
    dp[i] = 状态转移;
}
```

### 2. 二维 DP

```java
int[][] dp = new int[m + 1][n + 1];

for (int i = 1; i <= m; i++) {
    for (int j = 1; j <= n; j++) {
        dp[i][j] = 状态转移;
    }
}
```

### 3. 0-1 背包

```java
int[] dp = new int[capacity + 1];

for (int i = 0; i < n; i++) {
    for (int j = capacity; j >= weight[i]; j--) {
        dp[j] = Math.max(dp[j], dp[j - weight[i]] + value[i]);
    }
}
```

### 4. 完全背包

```java
int[] dp = new int[capacity + 1];

for (int i = 0; i < n; i++) {
    for (int j = weight[i]; j <= capacity; j++) {
        dp[j] = Math.max(dp[j], dp[j - weight[i]] + value[i]);
    }
}
```

## 八、图

### 1. DFS

```java
void dfs(int node) {
    if (visited[node]) return;
    visited[node] = true;

    for (int next : graph[node]) {
        dfs(next);
    }
}
```

### 2. BFS

```java
Queue<Integer> queue = new ArrayDeque<>();
queue.offer(start);
visited[start] = true;

while (!queue.isEmpty()) {
    int cur = queue.poll();
    for (int next : graph[cur]) {
        if (!visited[next]) {
            visited[next] = true;
            queue.offer(next);
        }
    }
}
```

### 3. 拓扑排序

```java
Queue<Integer> queue = new ArrayDeque<>();
for (int i = 0; i < n; i++) {
    if (indegree[i] == 0) {
        queue.offer(i);
    }
}

int count = 0;
while (!queue.isEmpty()) {
    int cur = queue.poll();
    count++;
    for (int next : graph[cur]) {
        if (--indegree[next] == 0) {
            queue.offer(next);
        }
    }
}

return count == n;
```

## 九、堆

适用：

- Top K。
- 合并 K 个有序链表。
- 数据流中位数。

小顶堆：

```java
PriorityQueue<Integer> heap = new PriorityQueue<>();
```

大顶堆：

```java
PriorityQueue<Integer> heap = new PriorityQueue<>((a, b) -> Integer.compare(b, a));
```

Top K 大：

```java
PriorityQueue<Integer> heap = new PriorityQueue<>();
for (int num : nums) {
    heap.offer(num);
    if (heap.size() > k) {
        heap.poll();
    }
}
return heap.peek();
```
