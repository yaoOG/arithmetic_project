
题目：给你二叉树的根节点 `root` ，返回其节点值的 **锯齿形层序遍历** 。（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

锯齿形层序遍历（Zigzag Level Order Traversal）是常规层序遍历的一个变种。它的核心逻辑依然是 **广度优先搜索 (BFS)**，但在将节点值存入结果集时，我们需要根据层数的奇偶性来决定填充的方向。

### 1. 核心思路

- **层序遍历基石**：使用队列 `Queue` 进行典型的 BFS。
    
- **方向控制**：维护一个布尔变量 `isOrderLeft`。
    
    - 如果为 `true`：从左向右放入容器（尾插）。
        
    - 如果为 `false`：从右向左放入容器（头插）。
        
- **双端队列的妙用**：在处理每一层的结果时，使用 `LinkedList` 作为双端队列（Deque），它支持 $O(1)$ 时间复杂度的 `addFirst()` 和 `addLast()`，这比反转整个列表更高效。
    

---

### 2. 代码实现 (Java)


```java
public List<List<Integer>> zigzagLevelOrder(TreeNode root) {  
    List<List<Integer>> result = new ArrayList<>();  
    if (root == null) return result;  
  
    Deque<TreeNode> queue = new ArrayDeque<>();  
    queue.offer(root);  
  
    // 控制遍历方向的标志位  
    boolean isOrderLeft = true;  
  
    while (!queue.isEmpty()) {  
        int size = queue.size();  
        // 使用 LinkedList 方便头尾操作  
        LinkedList<Integer> levelList = new LinkedList<>();  
  
        for (int i = 0; i < size; i++) {  
            TreeNode curr = queue.poll();  
            if (isOrderLeft) {  
                // 正常顺序：添加到末尾  
                levelList.addLast(curr.val);  
            } else {  
                // 反向顺序：添加到头部  
                levelList.addFirst(curr.val);  
            }  
            if (curr.left != null) queue.addLast(curr.left);  
            if (curr.right != null) queue.addLast(curr.right);  
        }  
        result.add(levelList);  
        // 每一层结束，反转方向  
        isOrderLeft = !isOrderLeft;  
    }  
  
    return result;  
}
```

---

### 3. 复杂度分析

- **时间复杂度**：$O(n)$。每个节点进出队列一次，且由于使用了 `LinkedList` 的头/尾插，每一层的操作也是线性的。
    
- **空间复杂度**：$O(n)$。最坏情况下，队列中会存储树中节点数最多的一层（通常是叶子节点层，约 $n/2$ 个节点）。
