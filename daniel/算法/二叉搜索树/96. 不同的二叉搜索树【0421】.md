题目：给你一个整数 `n` ，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 **二叉搜索树** 有多少种？返回满足题意的二叉搜索树的种数。

这是一个经典的组合数学问题，可以通过 **动态规划 (Dynamic Programming)** 或直接使用 **卡特兰数 (Catalan Number)** 来解决。

### 1. 解题思路：动态规划

给定 $n$ 个节点，我们可以选择其中任意一个数字 $i$ ($1 \le i \le n$) 作为根节点。

根据 **二叉搜索树 (BST)** 的性质：

- 左子树的所有节点值必须小于根节点 $i$，因此左子树由 $[1, i-1]$ 这 $i-1$ 个节点组成。
    
- 右子树的所有节点值必须大于根节点 $i$，因此右子树由 $[i+1, n]$ 这 $n-i$ 个节点组成。
    

设 $G(n)$ 为长度为 $n$ 的序列能构成的不同二叉搜索树的个数。

设 $f(i, n)$ 为以 $i$ 为根节点、序列长度为 $n$ 的不同二叉搜索树个数。

我们可以得出公式：

1. **总数是各节点作为根的情况之和**：$G(n) = \sum_{i=1}^{n} f(i, n)$
    
2. **以 $i$ 为根的数量等于左、右子树可能情况的乘积**：$f(i, n) = G(i-1) \times G(n-i)$
    

结合两者，得到状态转移方程：

$$G(n) = \sum_{i=1}^{n} G(i-1) \times G(n-i)$$

---

### 2. 算法实现 (Java)

作为一名后端架构师，你一定追求代码的简洁与高效。这里使用迭代的方式来实现：

```Java
public int numTrees(int n) {
    // dp[i] 表示 i 个节点能组成的 BST 种类数
    int[] dp = new int[n + 1];
    
    // 边界情况：空树或只有一个节点的树，只有 1 种形态
    dp[0] = 1;
    dp[1] = 1;
    
    // 从 2 个节点开始计算到 n
    for (int i = 2; i <= n; i++) {
        for (int j = 1; j <= i; j++) {
            // j 作为根节点，左边有 j-1 个节点，右边有 i-j 个节点
            dp[i] += dp[j - 1] * dp[i - j];
        }
    }
    
    return dp[n];
}
```

---

### 3. 数学解法：卡特兰数

实际上，上述递推公式定义的数列正是 **卡特兰数**。其通项公式为：

$$C_n = \frac{1}{n+1} \binom{2n}{n} = \frac{(2n)!}{(n+1)!n!}$$

**复杂度分析：**

- **时间复杂度**：动态规划方法为 $O(n^2)$。如果使用数学公式直接计算，可以达到 $O(n)$。
    
- **空间复杂度**：动态规划需要 $O(n)$ 的数组来存储状态。
    

这个问题在面试中经常作为考察 DP 思路的入门题。如果你在处理海量计费数据或构建复杂的索引树时，理解这种组合增长的规律也会很有帮助。
