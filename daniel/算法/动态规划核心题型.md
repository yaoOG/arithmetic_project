# 动态规划核心题型

## 一、先说结论

动态规划不是背公式，而是解决有重叠子问题和最优子结构的问题。

四步：

```text
1. 定义 dp 数组含义。
2. 写状态转移方程。
3. 明确初始化。
4. 确定遍历顺序。
```

面试时最重要的是讲清：

```text
dp[i] 代表什么，而不是直接写代码。
```

## 二、爬楼梯模型

问题：

```text
每次爬 1 或 2 阶，到第 n 阶有多少种方法。
```

定义：

```text
dp[i] = 爬到第 i 阶的方法数
```

转移：

```text
dp[i] = dp[i - 1] + dp[i - 2]
```

代码：

```java
public int climbStairs(int n) {
    if (n <= 2) return n;
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) {
        int c = a + b;
        a = b;
        b = c;
    }
    return b;
}
```

## 三、最大子数组和

定义：

```text
dp[i] = 以 nums[i] 结尾的最大子数组和
```

转移：

```text
dp[i] = max(nums[i], dp[i - 1] + nums[i])
```

代码：

```java
public int maxSubArray(int[] nums) {
    int dp = nums[0];
    int ans = nums[0];
    for (int i = 1; i < nums.length; i++) {
        dp = Math.max(nums[i], dp + nums[i]);
        ans = Math.max(ans, dp);
    }
    return ans;
}
```

关键：

```text
如果前面的连续和是负数，不如从当前元素重新开始。
```

## 四、打家劫舍

定义：

```text
dp[i] = 偷到第 i 间房时能获得的最大金额
```

转移：

```text
dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])
```

代码：

```java
public int rob(int[] nums) {
    int prev2 = 0, prev1 = 0;
    for (int num : nums) {
        int cur = Math.max(prev1, prev2 + num);
        prev2 = prev1;
        prev1 = cur;
    }
    return prev1;
}
```

## 五、最长递增子序列

### DP 解法

定义：

```text
dp[i] = 以 nums[i] 结尾的最长递增子序列长度
```

转移：

```text
如果 nums[j] < nums[i]，dp[i] = max(dp[i], dp[j] + 1)
```

代码：

```java
public int lengthOfLIS(int[] nums) {
    int n = nums.length;
    int[] dp = new int[n];
    Arrays.fill(dp, 1);
    int ans = 1;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                dp[i] = Math.max(dp[i], dp[j] + 1);
            }
        }
        ans = Math.max(ans, dp[i]);
    }
    return ans;
}
```

复杂度：

```text
O(n^2)
```

进阶可以用贪心 + 二分做到 O(nlogn)。

## 六、0-1 背包

问题：

```text
每个物品只能选一次，在容量限制下最大价值。
```

定义：

```text
dp[j] = 容量为 j 时的最大价值
```

转移：

```text
dp[j] = max(dp[j], dp[j - weight[i]] + value[i])
```

代码：

```java
public int knapsack(int[] weight, int[] value, int capacity) {
    int[] dp = new int[capacity + 1];
    for (int i = 0; i < weight.length; i++) {
        for (int j = capacity; j >= weight[i]; j--) {
            dp[j] = Math.max(dp[j], dp[j - weight[i]] + value[i]);
        }
    }
    return dp[capacity];
}
```

为什么倒序：

```text
倒序保证每个物品只被使用一次。
```

## 七、完全背包

每个物品可以选无限次。

代码：

```java
public int completeKnapsack(int[] weight, int[] value, int capacity) {
    int[] dp = new int[capacity + 1];
    for (int i = 0; i < weight.length; i++) {
        for (int j = weight[i]; j <= capacity; j++) {
            dp[j] = Math.max(dp[j], dp[j - weight[i]] + value[i]);
        }
    }
    return dp[capacity];
}
```

为什么正序：

```text
正序允许当前物品被重复使用。
```

## 八、编辑距离

定义：

```text
dp[i][j] = word1 前 i 个字符转换成 word2 前 j 个字符的最少操作数
```

转移：

```text
如果 word1[i-1] == word2[j-1]：
dp[i][j] = dp[i-1][j-1]

否则：
dp[i][j] = min(
    dp[i-1][j] + 1,    删除
    dp[i][j-1] + 1,    插入
    dp[i-1][j-1] + 1   替换
)
```

## 九、常见错误

- dp 定义不清。
- 初始化漏掉边界。
- 遍历顺序错。
- 0-1 背包和完全背包内层方向写反。
- 子数组和子序列混淆。
- 最大值初始化为 0 导致全负数错误。

## 十、面试表达模板

```text
我会先定义 dp[i] 的业务含义，再根据最后一步选择推导状态转移。
比如最大子数组和，dp[i] 表示以 i 结尾的最大和，因此只有两个选择：接上前面的子数组，或者从当前元素重新开始。
然后处理初始化和遍历顺序，最后再考虑空间压缩。
```
