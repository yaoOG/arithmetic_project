### 一、自定义类加载器并不复杂
只需继承 java.lang.ClassLoader 类，并重写它的 findClass(String name) 方法即可。

### 二、 java.lang.ClassLoader 类关键方法

- loadClass(String name, boolean resolve)：实现了双亲委派模型的逻辑。一般**不建议重写**这个方法，除非你想破坏双亲委派模型。
- findClass(String name)：**推荐的重写方法**。在 loadClass 内部被调用。它的职责是根据指定的类名，找到类的二进制字节流，然后调用 defineClass 方法将其转换为一个 Class 对象。
- defineClass(byte[] b, int off, int len)：**final方法，不可重写**。负责将字节数组转换为 Class 对象。这是JVM层面的魔法。

### 三、示例
```java
public class MyClassLoader extends ClassLoader {  
  
    // 指定类文件所在的根目录  
    private String classPath;  
  
    public MyClassLoader(String classPath) {  
        // 指定父加载器为 AppClassLoader        super(ClassLoader.getSystemClassLoader());  
        this.classPath = classPath;  
    }  
  
    @Override  
    protected Class<?> findClass(String name) throws ClassNotFoundException {  
        // 1. 根据类名，转换为文件系统路径  
        String fileName = name.replace('.', File.separatorChar) + ".class";  
        File classFile = new File(classPath, fileName);  
  
        // 2. 尝试读取该文件的字节码  
        try (FileInputStream fis = new FileInputStream(classFile);  
             ByteArrayOutputStream bos = new ByteArrayOutputStream()) {  
  
            int bufferSize = 1024;  
            byte[] buffer = new byte[bufferSize];  
            int bytesNumRead;  
            while ((bytesNumRead = fis.read(buffer)) != -1) {  
                bos.write(buffer, 0, bytesNumRead);  
            }  
            byte[] classData = bos.toByteArray();  
  
            // 3. 调用 defineClass 将字节数组转换为 Class 对象  
            return defineClass(name, classData, 0, classData.length);  
  
        } catch (IOException e) {  
            throw new ClassNotFoundException("Class " + name + " not found.", e);  
        }  
    }  
  
    // 测试  
    public static void main(String[] args) throws Exception {  
        // 创建一个自定义类加载器，从 /tmp/classes 目录加载类  
        MyClassLoader myLoader = new MyClassLoader("/tmp/classes");  
  
        // 加载一个名为 com.example.Test 的类  
        Class<?> clazz = myLoader.loadClass("com.example.Test");  
  
        // 创建实例并调用方法  
        Object instance = clazz.newInstance();  
        clazz.getMethod("sayHello").invoke(instance);  
    }  
}
```
